# Apple Developer News キャッチアップ: 2026-06-10

> 取得日: 2026-06-10
> ソース: [Apple Developer News](https://developer.apple.com/jp/news/)

## 今回の注目ポイント

WWDC26（2026年6月8日）に合わせて重要な更新が集中しています。**Apple Developer Program 使用許諾契約と App Review ガイドラインが改訂**され、デベロッパはサインインして同意が必要です。また iOS 27 / iPadOS 27 / macOS 27 で導入される**「許容時間（Time Allowance）」**に伴い、App Store Connect でのアプリ情報・ソーシャルメディア機能の申告が 2026年9月から義務化されます。あわせて、テキサス州 SB 2420 への対応（年齢認証 API）と、ベルリンへの Apple Developer Center 拡大が発表されました。

## 記事一覧

### Appleデベロッパ向けの最新情報（Find out what's new for Apple developers）
- **URL**: https://developer.apple.com/jp/news/?id=8rgqj83s
- **公開日**: 2026-06-08
- **要約**: WWDC26 で、言語・フレームワーク・ツール・サービスにわたる大幅な強化と、各 Apple プラットフォーム向けの最新アップデートが発表。洗練されたデザイン、強力な Apple Intelligence の機能、新しい AI 開発フレームワークを含む新 SDK が提供される。

### 「許容時間」の導入（Introducing Time Allowances）
- **URL**: https://developer.apple.com/jp/news/?id=0d2gpmml
- **公開日**: 2026-06-08
- **要約**: iOS 27 / iPadOS 27 / macOS 27 以降で「許容時間（Time Allowance）」が導入され、保護者がエンターテインメント・ゲーム・ソーシャルメディアなどのカテゴリ別に子どものアプリ利用時間を管理できる。デベロッパは App Store Connect でアプリ情報やソーシャルメディア機能の有無を申告する必要があり、2026年9月から義務化される。

### Apple Developer Program使用許諾契約およびApp Reviewガイドラインが更新されました（Updated Apple Developer Program License Agreement and App Review Guidelines now available）
- **URL**: https://developer.apple.com/jp/news/?id=a233fmpw
- **公開日**: 2026-06-08
- **要約**: 新機能やポリシーに合わせて使用許諾契約と App Review ガイドラインが改訂。Sensitive Content Analysis や Trust Insights など新フレームワークの使用要件の明確化、AI・機械学習テクノロジーに関する規定の追加、子どもとティーンエイジャーの安全ガイダンスの改訂などが含まれる。デベロッパはサインインして更新された契約に同意する必要がある。

### Update for Apps Distributed in Texas
- **URL**: https://developer.apple.com/jp/news/?id=sg176nne
- **公開日**: 2026-06-03
- **要約**: テキサス州法 SB 2420 の年齢認証要件に対応するための更新。テキサス州で配布されるアプリのデベロッパは、Declared Age Range API および Significant Change API を使用して年齢認証への対応を行う必要がある。

### 新たなApple Developer Centerをベルリンに開設（Apple Developer Centers are expanding to Berlin）
- **URL**: https://developer.apple.com/jp/news/?id=f0jfy9py
- **公開日**: 2026-06-03
- **要約**: ヨーロッパ初となる Apple Developer Center（5か所目）をベルリンに開設。セッション・ワークショップ・1対1のアポイントメント・ラボスペースを対面で提供し、Apple のエキスパートが実践的なサポートを多言語で行う。
