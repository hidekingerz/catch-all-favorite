## 今週の注目ポイント

今週最大のトピックは **pnpm v11.0.0 RC 0** のリリース。Pure ESM への移行と Node.js 18〜21 のサポート終了に加え、SBOM 生成や `.npmrc` 設定制限といったサプライチェーンセキュリティ強化が盛り込まれています。ビルドツール関連では **webpack 5.106** が CSS Modules のランタイムスタイル注入や Plugin Validation Hook、webpack-cli 7.0.0 でのネイティブ TypeScript サポートを追加。ブラウザエンジンでは Rust 製の **Servo v0.1.0** が crates.io で公開され、Embedding API を備えた組み込み可能なレンダリングエンジンとして新たな一歩を踏み出しました。React / Next.js エコシステムでは **CVE-2026-23869**（RSC の DoS 脆弱性）が公表されているため、利用プロジェクトは早急な影響確認が必要です。
