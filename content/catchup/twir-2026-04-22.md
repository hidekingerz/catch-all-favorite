## React
### React Email 6.0
- **URL**: https://resend.com/blog/react-email-6
- **要約**: パッケージの統合による断片化の解消、埋め込み可能なエディタ、メールクライアント間での信頼性の高いHTML描画、高速なHMRによるテンプレート切り替えなどを含むメジャーリリース。

- **URL**: https://tsrx.dev/
- **要約**: 宣言的UIのためのTypeScript言語拡張。JSXの精神的後継で、元Reactコアチームメンバーの Dominic Gannaway が作成。React/Solid/Rippleにコンパイル可能なフレームワーク非依存のアプローチ。

- **URL**: https://github.com/TanStack/store/releases/tag/%40tanstack/react-store%400.11.0

### ESLint Plugin React Hooks 7.1
- **URL**: https://github.com/facebook/react/releases/tag/eslint-plugin-react-hooks%407.1.0

### Lingui 6.0
- **URL**: https://lingui.dev/blog/2026/04/22/announcing-lingui-6.0
- **要約**: ReactおよびReact Native向けの国際化フレームワーク。技術的な改善とモダナイゼーションが施された6.0リリース。

### React Hook Form 7.73
- **URL**: https://github.com/react-hook-form/react-hook-form/releases/tag/v7.73.1
