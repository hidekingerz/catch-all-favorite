# JSer.info #775 キャッチアップ: 2026-07-05のJS

> 投稿日: 2026-07-05
> ソース: [2026-07-05のJS: Deno 2.9、Vite 8.1、ES2026](https://jser.info/2026/07/05/deno-2.9-vite-8.1-es2026/)

## 今週の注目ポイント

Deno 2.9 がリリースされ、Web技術でデスクトップアプリを作れる実験的機能「deno desktop」や、npm/pnpm/yarn/Bun のロックファイルから `deno.lock` を生成する機能が追加された。新しいnpmパッケージバージョンの即時インストールを避ける `min-release-age=24h` がデフォルト有効になっている点も注目。Vite 8.1 は開発時バンドルの実験的フラグ `--experimental-bundle` や Chunk Import Map、Wasm ESM 統合を導入した。そして 2026年6月30日に ECMAScript 2026 が正式承認され、`Array.fromAsync`、`Uint8Array` の Base64 変換メソッド、`Iterator.concat()`、`Math.sumPrecise()`、`Error.isError()`、`Map.prototype.getOrInsert()` などが標準化された。

## ヘッドライン

### Node.js — Node.js 24.18.0 (LTS)
- **URL**: https://nodejs.org/en/blog/release/v24.18.0
- **タグ**: nodejs ReleaseNote
- **要約**: `Buffer.poolSize` のデフォルトを64 KiBに変更。Web Crypto に TurboSHAKE と KangarooTwelve を追加し、`http.writeInformation()` や `crypto.diffieHellman()` の強化を含む。

### pnpm 11.9
- **URL**: https://pnpm.io/blog/releases/11.9
- **タグ**: pnpm ReleaseNote
- **要約**: integrityフィールドを持たないtarballに対してレジストリのチェックサムを計算するようになった。`pnpm sbom --exclude-peers` や `pnpm audit --fix` の出力フォーマット改善を含む。

### Announcing Rspack 2.1
- **URL**: https://rspack.rs/blog/announcing-2-1
- **タグ**: rspack ReleaseNote
- **要約**: `builtin:swc-loader` での React Compiler サポート、`import.meta.glob` サポート、永続キャッシュの自動削除、TypeScript 7 の型チェック統合を追加。

### Deno 2.9
- **URL**: https://deno.com/blog/v2.9
- **タグ**: deno ReleaseNote
- **要約**: 実験的な `deno desktop`、npmロックファイルの読み込み、`deno list`/`link`/`unlink` コマンド、新しいtest/compile/formatフラグ、`trust-policy=no-downgrade` を導入。

### Prettier 3.9
- **URL**: https://prettier.io/blog/2026/06/27/3.9.0
- **タグ**: JavaScript Tools ReleaseNote TypeScript css
- **要約**: Markdownパーサーを `micromark` v4、YAMLを `yaml` v2 にアップグレード。GraphQL.js v17対応、Rustベースの Flow パーサー、`import ... assert` サポートの削除を含む。

### Vite 8.1 is out!
- **URL**: https://vite.dev/blog/announcing-vite8-1
- **タグ**: vite ReleaseNote
- **要約**: 実験的な `--experimental-bundle`、Chunk Import Map、Wasm ESM 統合、`import.meta.glob` の `caseSensitive` オプション、`html.additionalAssetSources` を追加。

### Release 1.0.0 - Basecoat
- **URL**: https://github.com/hunvreus/basecoat/releases/tag/1.0.0
- **タグ**: JavaScript css HTML library ReleaseNote
- **要約**: React/Radix のランタイム依存なしで動く Tailwind CSS + shadcn/ui互換のコンポーネントライブラリ。CDN、npm、テンプレート経由で利用可能。

### Next.js 16.3: Instant Navigations
- **URL**: https://nextjs.org/blog/next-16-3-instant-navigations
- **タグ**: Next.js React ReleaseNote
- **要約**: `cacheComponents` と `partialPrefetching` による実験的な Instant Navigations を含むプレビューリリース。ルート遷移の高速化を狙う。

### Release ES2026
- **URL**: https://github.com/tc39/ecma262/releases/tag/es2026
- **タグ**: ECMAScript JavaScript ReleaseNote
- **要約**: 2026年6月30日に ES2026 が正式承認。`Array.fromAsync`、`Uint8Array` の Base64 メソッド、`Iterator.concat()`、`Math.sumPrecise()`、`Error.isError()`、`Map.prototype.getOrInsert()` を含む。

### Announcing Vite+ Beta
- **URL**: https://voidzero.dev/posts/announcing-vite-plus-beta
- **タグ**: vite CLI Tools ReleaseNote
- **要約**: `vp run` のキャッシュ改善、`vp migrate` のカバレッジ拡大、プロキシとカスタムCAのHTTP処理サポートを追加。
