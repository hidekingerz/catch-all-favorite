## 今週の注目ポイント

Go製コンパイラの TypeScript 7.0 RC が `typescript@rc` として公開され、`@typescript/typescript6` との併用も可能になった。Babel 8.0.0 と React Router v8 はいずれも ESM-only 配布へ移行し、デフォルトの ES5 変換を廃止するなど大きな破壊的変更を伴うメジャーリリースとなっている。ツール周りでは Biome v2.5、pnpm 11.7、Playwright v1.61.0、Safari 27 Beta が出揃った。

---

## ヘッドライン

### Announcing TypeScript 7.0 RC
- **URL**: https://devblogs.microsoft.com/typescript/announcing-typescript-7-0-rc/
- **タグ**: TypeScript, ReleaseNote, go
- **要約**: Go製コンパイラを `typescript@rc` としてリリース。`@typescript/typescript6` との併用が可能。`--checkers`/`--builders`/`--singleThreaded` オプションの追加と `--watch` の改善。`strict`/`module`/`target` のデフォルト変更、`target: es5` と `baseUrl` オプションの削除。

### Babel 8.0.0
- **URL**: https://babeljs.io/blog/2026/06/16/8.0.0/
- **タグ**: babel, ReleaseNote, ECMAScript, nodejs, polyfill
- **要約**: ESM-only 配布となり Node.js 24 以上が必須。`@babel/preset-env` はデフォルトで ES5 へ変換しなくなり ES2023 をターゲットに。`loose`/`spec` オプションを削除（`assumptions` へ移行）、`corejs`/`useBuiltIns` を削除（`babel-plugin-polyfill-corejs3` を使用）。Babel 7 は 2027年6月までセキュリティサポート。

### React Router v8
- **URL**: https://remix.run/blog/react-router-v8
- **タグ**: React, ReleaseNote
- **要約**: Node.js 22.22.0 以上・React 19.2.7 以上・Vite 7 以上が必須。ESM-only 配布。Future flags がデフォルトで有効化。`react-router-dom`、meta API の `data` 引数、Cloudflare vite プラグインを削除。React Router v6 と Remix v2 は EOL、v7 はセキュリティ更新のみ。

### Dependabot version updates now support the Deno ecosystem
- **URL**: https://github.blog/changelog/2026-06-09-dependabot-version-updates-now-support-the-deno-ecosystem/
- **タグ**: github, deno, npm, package, Actions, news
- **要約**: Dependabot の version updates が Deno エコシステムをサポートするようになった。

### WASI 0.3
- **URL**: https://bytecodealliance.org/articles/WASI-0.3
- **タグ**: WebAssembly, ReleaseNote
- **要約**: `stream<T>`/`future<T>` や Canonical ABI の `async` によるネイティブな非同期サポートを追加。ホストが共有イベントループを管理する方式に。従来の `start-foo`/`finish-foo`/`subscribe` パターンを `async func` に置き換える。

### Biome v2.5
- **URL**: https://biomejs.dev/blog/biome-v2-5/
- **タグ**: biome, ReleaseNote, ESLint, css
- **要約**: クロスファイルの CSS 解析（`noUndeclaredClasses`/`noUnusedClasses`）を含む 500 個の lint ルールを追加。GritQL プラグインがファイル単位の適用とコード修正をサポート。`delimiterSpacing` オプション、`--watch` サポート、Language Server の go-to-definition、AI エージェント向けの `concise` レポーターを追加。

### pnpm 11.7
- **URL**: https://pnpm.io/blog/releases/11.7
- **タグ**: pnpm, ReleaseNote
- **要約**: 読み取り専用ファイルシステムストア向けの `frozenStore` 設定を追加。同一レジストリ URL 配下でのスコープ固有の認証トークンをサポート。依存解決に Rust製の Pacquet を使い始めた。

### Playwright v1.61.0
- **URL**: https://github.com/microsoft/playwright/releases/tag/v1.61.0
- **タグ**: playwright, testing, ReleaseNote
- **要約**: `browserContext.credentials` による WebAuthn パスキーをサポート。Web Storage API（`page.localStorage`/`page.sessionStorage`）を追加。`apiResponse.securityDetails()`/`apiResponse.serverAddr()`、`artifactsDir` オプション、`-G` オプションを追加。

### News from WWDC26: WebKit in Safari 27 beta
- **URL**: https://webkit.org/blog/17967/news-from-wwdc26-webkit-in-safari-27-beta/
- **タグ**: safari, browser, ReleaseNote, css, JavaScript
- **要約**: `<select>` のカスタマイズ向け CSS `appearance: base-select`、Scroll Anchoring、`position-anchor: normal` を追加。Top-Level Await の問題を修正し WebAssembly JSPI をサポート。Service Worker Static Routing API、`ReadableStream.from()`、Stream の `postMessage()` 転送を追加。

## アーティクル

### Upcoming breaking changes for npm v12
- **URL**: https://github.blog/changelog/2026-06-09-upcoming-breaking-changes-for-npm-v12/
- **タグ**: npm, security, CLI, article
- **要約**: preinstall/install/postinstall などのライフサイクルスクリプトは `npm approve-scripts` による明示的な承認が必須に。`--allow-git`/`--allow-remote` はデフォルトで `none` となり、許可なしでは Git 依存やリモート URL をブロック。2026年7月のリリース予定。

### Environment variables in repository .npmrc
- **URL**: https://pnpm.io/blog/2026/06/11/env-variables-in-repository-npmrc
- **タグ**: pnpm, security, npm, article
- **要約**: 悪意ある抽出を防ぐため、リポジトリの `.npmrc` では環境変数を展開しなくなった。v10.34.2 以上・v11.5.3 以上に適用。ユーザーレベルの `~/.npmrc` と `PNPM_CONFIG_NPMRC_AUTH_FILE` のファイルは引き続き変数を展開する。

### Introducing the MDN MCP server
- **URL**: https://developer.mozilla.org/en-US/blog/introducing-mdn-mcp-server/
- **タグ**: mdn, MCP, document, browser, AI, article
- **要約**: MCP 経由で MDN の検索・ドキュメント・ブラウザ互換性データにアクセスできるサーバー。LLM やエディタとの統合を可能にする。クエリデータをログに記録する実験的サービス。
