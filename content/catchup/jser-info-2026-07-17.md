## 今週の注目ポイント

TypeScript 7.0 が正式リリースされ、Go で書かれたネイティブ実装の `tsc` が npm の typescript パッケージ経由で配布されるようになった。`target: es5` や `baseUrl` などの非推奨だったオプションはエラーになり、並列処理を制御する `--checkers`/`--builders`/`--singleThreaded` や再設計された `--watch` が加わった。ただしプログラマティックAPIは未提供のため、Vue など tsc を組み込みで利用するツールには影響がある。npm 12 では `allowScripts` がデフォルトで無効化され、`--allow-git`/`--allow-remote` のデフォルトも `none` になった。2FAをバイパスする Granular Access Token は 2026年8月から制限され、2027年1月には直接publishが無効化される予定。Next.js は毎月の定期セキュリティリリース方針を発表し、事前告知付きのスケジュール公開と緊急時のアドホックパッチを併用する。最初のリリースは 2026年7月20日、対象は 16.2 と 15.5。

## ヘッドライン

### Release pnpm 11.10
- **URL**: https://github.com/pnpm/pnpm/releases/tag/v11.10.0
- **タグ**: pnpm ReleaseNote
- **要約**: `pnpm issues`/`pnpm prefix` コマンドの追加、`pnpm_config__auth` 環境変数のサポート、`self-update` の pnpm v12 対応、`pnpm up -r` や `pnpm pack-app` の修正を含む。

### Chrome 150
- **URL**: https://developer.chrome.com/release-notes/150
- **タグ**: Chrome browser css DOM WebGPU ReleaseNote
- **要約**: CSS の `AccentColor`/`AccentColorText`、polygon の角丸、`text-fit` をサポート。`url()` に cross-origin/integrity/referrer-policy 修飾子を追加。DOM の `focusgroup` 属性、`popover=hint` の挙動変更、WebGPU の `setImmediateData()` と PWA のオリジン移行を含む。

### Electron 43
- **URL**: https://www.electronjs.org/blog/electron-43-0
- **タグ**: Electron ReleaseNote
- **要約**: Chromium 150.0.7871.46、Node.js v24.17.0、V8 15.0 へ更新。ファイルダウンロードダイアログのデフォルトがダウンロードフォルダに。`WebContents.clone()`、macOS の `Notification.remove()`/`removeAll()`、`globalShortcut.setSuspended()` を追加し、startup snapshot のパフォーマンスを改善。

### Announcing TypeScript 7.0
- **URL**: https://devblogs.microsoft.com/typescript/announcing-typescript-7-0/
- **タグ**: TypeScript ReleaseNote
- **要約**: Go 実装のネイティブ `tsc` を npm の typescript パッケージ経由で配布。非推奨だった `target: es5`/`baseUrl` はエラーに。並列処理制御と再設計された `--watch` を追加。プログラマティックAPIは未提供で、Vue などのツールに影響する。

### Better Auth is joining Vercel
- **URL**: https://better-auth.com/blog/better-auth-joins-vercel
- **タグ**: TypeScript vercel news nodejs
- **要約**: Vercel が TypeScript 製の認証ライブラリ Better Auth を買収。MIT ライセンスと開発の継続性を維持。Agent Auth Protocol の開発や、エージェントID管理での Vercel Connect・Eve との統合を計画している。

### npm install-time security and GAT bypass2fa deprecation
- **URL**: https://github.blog/changelog/2026-07-08-npm-install-time-security-and-gat-bypass2fa-deprecation/
- **タグ**: npm security ReleaseNote
- **要約**: npm v12 は `allowScripts` をデフォルト無効化し、`--allow-git`/`--allow-remote` のデフォルトを `none` に。Granular Access Token は 2026年8月からアカウント操作での2FAバイパスを失い、2027年1月に直接publishが無効化される。Trusted Publishing または Staged Publishing を推奨。

### Node.js 26.5.0 (Current)
- **URL**: https://nodejs.org/en/blog/release/v26.5.0
- **タグ**: nodejs ReleaseNote
- **要約**: `Blob.prototype.textStream()`、`--experimental-import-text`、`ReadableStreamTee` を追加。`perf_hooks.monitorEventLoopDelay` に `samplePerIteration` オプションを追加。
