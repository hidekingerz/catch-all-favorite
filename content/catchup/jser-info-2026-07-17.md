# JSer.info #776 キャッチアップ: 2026-07-17のJS

> 投稿日: 2026-07-17
> ソース: [2026-07-17のJS: TypeScript 7.0、npm 12、Next.jsのセキュリティリリース方針](https://jser.info/2026/07/17/typescript-7.0-npm-12-next.js/)

## 今週の注目ポイント

今週は大型リリースが集中。TypeScript 7.0 は Go に移植されたネイティブ `tsc` が npm パッケージ経由で利用可能になり、並列処理制御オプションや LSP 対応が追加された（プログラマティックAPIは未公開のため Vue 等のツールは非互換の可能性あり）。npm 12 ではライフサイクルスクリプトの `allowScripts` がデフォルト無効化され、GAT の 2FA バイパス廃止スケジュールが提示されるなどセキュリティ強化が進む。Next.js は毎月の定例セキュリティリリース方針を発表し、影響バージョンと深刻度を事前告知する運用へ移行する。

## ヘッドライン

### pnpm 11.10
- **URL**: https://github.com/pnpm/pnpm/releases/tag/v11.10.0
- **タグ**: pnpm, ReleaseNote
- **要約**: 新コマンド `pnpm issues` と `pnpm prefix` を追加。`pnpm_config__auth` 環境変数に対応し、`pnpm self-update` が pnpm v12 と互換に。`pnpm up -r` や `pnpm pack-app` のバグ修正を含む。

### Chrome 150
- **URL**: https://developer.chrome.com/release-notes/150
- **タグ**: Chrome, browser, css, DOM, WebGPU, ReleaseNote
- **要約**: CSS では `AccentColor`/`AccentColorText`、`polygon()` の丸め、`text-fit`、`url()` 修飾子（`cross-origin()`/`integrity()`/`referrer-policy()`）を追加。DOM の `focusgroup` 属性、`popover=hint` の挙動変更、scroll メソッドの Promise 返却、WebGPU の `setImmediateData()` などが導入された。

### Electron 43
- **URL**: https://www.electronjs.org/blog/electron-43-0
- **タグ**: Electron, ReleaseNote
- **要約**: Chromium 150、Node.js v24.17.0、V8 15.0 に更新。ファイルダウンロードダイアログが既定でダウンロードフォルダを開くように。`WebContents.clone()`、macOS の `Notification.remove()`/`removeAll()`、`globalShortcut.setSuspended()` を追加し、起動スナップショットや V8 バイトコードキャッシュで性能改善。

### TypeScript 7.0
- **URL**: https://devblogs.microsoft.com/typescript/announcing-typescript-7-0/
- **タグ**: TypeScript, ReleaseNote
- **要約**: Go に移植したネイティブ `tsc` を npm パッケージで提供。非推奨設定がエラー化。`--checkers`/`--builders`/`--singleThreaded` で並列処理を制御でき、`--watch` は Parcel のファイルウォッチャーで再構築。LSP に対応するが、プログラマティックAPIは未公開のため Vue 等のツールは非互換の可能性がある。

### Better Auth Joins Vercel
- **URL**: https://better-auth.com/blog/better-auth-joins-vercel
- **タグ**: TypeScript, vercel, news, nodejs
- **要約**: Vercel が TypeScript 製認証ライブラリ Better Auth を買収。MIT ライセンスは維持。Agent Auth Protocol の開発を計画し、Vercel Connect や Eve と統合してエージェントIDの管理を行う予定。

### npm v12 のインストール時セキュリティと GAT bypass2fa 廃止
- **URL**: https://github.blog/changelog/2026-07-08-npm-install-time-security-and-gat-bypass2fa-deprecation/
- **タグ**: npm, security, ReleaseNote
- **要約**: `allowScripts` を既定で無効化し、`--allow-git`/`--allow-remote` の既定を `none` に。GAT の 2FA バイパス廃止（2026年8月にアカウント/パッケージ操作、2027年1月に直接publish をブロック）を予告し、Trusted Publishing または Staged Publishing への移行を推奨。

### Node.js 26.5.0
- **URL**: https://nodejs.org/en/blog/release/v26.5.0
- **タグ**: nodejs, ReleaseNote
- **要約**: `Blob.prototype.textStream()` を追加。`--experimental-import-text` フラグ、`ReadableStreamTee`、`perf_hooks.monitorEventLoopDelay` の `samplePerIteration` オプションを導入。

## アーティクル

### Cloudflare Workers に Cache レイヤーを追加
- **URL**: https://blog.cloudflare.com/workers-cache/
- **タグ**: cloudflare, JavaScript, article
- **要約**: Workers の前段にキャッシュレイヤーを追加。`wrangler.jsonc` の `cache.enabled` で有効化し、`Cache-Control`/`Vary` に対応。`ctx.cache.purge()` でタグやパス単位のパージが可能。

### Next.js のセキュリティリリースプログラム
- **URL**: https://nextjs.org/blog/next-security-release-program
- **タグ**: Next.js, security, article
- **要約**: 毎月の事前告知付きセキュリティリリースを実施。緊急脆弱性は別途即時対応。2026年7月20日を初回目標とし、Next.js 16.2 と 15.5 向けのパッチを予定。

## ソフトウェア、ツール、ライブラリ関係

### ttsc (TypeScript-Go Toolchain)

- **タグ**: TypeScript, Tools, nodejs, lint

### ant (JavaScript Runtime)
- **URL**: https://github.com/theMackabu/ant
- **タグ**: JavaScript, C, TypeScript, Tools
- **要約**: C で実装された小さな JavaScript ランタイム。WinterTC の Minimum Common API 互換を目指す。

### Wordgard
- **URL**: https://wordgard.net/
- **タグ**: JavaScript, TypeScript, editor, library
- **要約**: ProseMirror/CodeMirror の作者によるリッチテキストエディタライブラリ。スキーマベースの文書構造と、CodeMirror 的な拡張機構を持つ。

### virtua
- **URL**: https://github.com/inokawa/virtua/
- **タグ**: React, Vue, Solid, Svelte, library
- **要約**: 約3kB の仮想スクロールコンポーネント。可変サイズ、水平スクロール、ウィンドウスクロール、グリッド表示に対応し、React/Vue/Solid/Svelte で利用できる。

### httpxy
- **URL**: https://github.com/unjs/httpxy
- **タグ**: nodejs, HTTP, WebSocket, library
- **要約**: Node.js 向けの HTTP/WebSocket プロキシ。`proxyFetch`、`proxyUpgrade`、`createProxyServer` を提供し、Unix Socket、`changeOrigin`、`xfwd`、ヘッダー書き換えに対応。
