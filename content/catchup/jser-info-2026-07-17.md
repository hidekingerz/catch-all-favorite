## 今週の注目ポイント

TypeScript 7.0 が正式リリースされ、Go で書き直されたネイティブ `tsc` コンパイラが npm 経由で利用可能になった。`target: es5` や `baseUrl` などの非推奨設定がエラーになり、`--checkers`/`--builders`/`--singleThreaded` による並列処理制御や刷新された `--watch`、LSP 対応が追加された一方、プログラマティック API は未公開のためツール側の対応はまだ限定的。npm 12 は `allowScripts` をデフォルト無効化し、2FA をバイパスする Granular Access Token を段階的に廃止する方針を打ち出した。さらに Next.js が毎月のセキュリティリリース方針を発表し、影響を受けるバージョンと深刻度を事前告知する運用へ移行する。

## ヘッドライン

### Release pnpm 11.10 · pnpm/pnpm
- **URL**: https://github.com/pnpm/pnpm/releases/tag/v11.10.0
- **タグ**: pnpm ReleaseNote
- **要約**: 新しい `pnpm issues`/`pnpm prefix` コマンド、`pnpm_config__auth` 環境変数のサポート、`self-update` の pnpm v12 互換対応を追加。無関係なパッケージが更新される問題やパス検証の不具合を修正。

### Chrome 150 | Release notes
- **URL**: https://developer.chrome.com/release-notes/150
- **タグ**: Chrome browser css DOM WebGPU ReleaseNote
- **要約**: CSS の `AccentColor`/`AccentColorText`、`polygon()` の丸め、`text-fit`、`url()` 修飾子（`cross-origin()`/`integrity()`/`referrer-policy()`）に対応。DOM の `focusgroup` 属性、`popover=hint` の挙動変更、Promise を返す scroll メソッド、WebGPU の `setImmediateData()`、PWA のオリジン移行、Origin Trial の `performance.getSpeculations()` を含む。

### Electron 43 | Electron
- **URL**: https://www.electronjs.org/blog/electron-43-0
- **タグ**: Electron ReleaseNote
- **要約**: Chromium 150.0.7871.46、Node.js v24.17.0、V8 15.0 に更新。ファイルダイアログのデフォルトが Downloads フォルダに変更。`WebContents.clone()`、macOS の `Notification.remove()`/`removeAll()`、`globalShortcut.setSuspended()` を追加。起動スナップショットと V8 バイトコードキャッシュでパフォーマンスを改善。

### Announcing TypeScript 7.0 - TypeScript
- **URL**: https://devblogs.microsoft.com/typescript/announcing-typescript-7-0/
- **タグ**: TypeScript ReleaseNote
- **要約**: Go に移植したネイティブ `tsc` を npm でリリース。非推奨設定はエラーになる。並列処理の制御、Parcel のファイルウォッチャーを使った刷新済み `--watch`、LSP 対応を追加。ただしプログラマティック API は未公開で、Vue など一部ツールとの互換性は限定的。

### Better Auth is joining Vercel
- **URL**: https://better-auth.com/blog/better-auth-joins-vercel
- **タグ**: TypeScript vercel news nodejs
- **要約**: Vercel が TypeScript 製の認証ライブラリ Better Auth を買収。MIT ライセンスで開発を継続。Agent Auth Protocol の開発や、Vercel Connect・Eve への統合を予定。

### npm install-time security and GAT bypass2fa deprecation
- **URL**: https://github.blog/changelog/2026-07-08-npm-install-time-security-and-gat-bypass2fa-deprecation/
- **タグ**: npm security ReleaseNote
- **要約**: npm v12 は `allowScripts` をデフォルト無効化し、`--allow-git`/`--allow-remote` のデフォルトを `none` に。2FA をバイパスする Granular Access Token は 2026年8月に権限昇格を失い、2027年1月には直接 publish が制限される。Trusted Publishing / Staged Publishing への移行を推奨。

### Node.js — Node.js 26.5.0 (Current)
- **URL**: https://nodejs.org/en/blog/release/v26.5.0
- **タグ**: nodejs ReleaseNote
- **要約**: `Blob.prototype.textStream()`、`--experimental-import-text`、`ReadableStreamTee` を追加。`perf_hooks.monitorEventLoopDelay` に `samplePerIteration` オプションを追加。
