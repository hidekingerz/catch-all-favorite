## 今週の注目ポイント

TypeScript 7.0 が正式リリースされ、Go で書かれたネイティブ版 `tsc` コンパイラが npm で配布されるようになった。`target: es5` や `baseUrl` などの非推奨設定はエラーになり、`--checkers`/`--builders`/`--singleThreaded` で並列処理を制御できる。一方、Program 向け API はまだ公開されておらず、Vue など一部ツールとの互換性には注意が必要。npm 12 はライフサイクルスクリプトをデフォルト無効化し、2FA をバイパスする Granular Access Token を段階的に廃止するなどインストール時のセキュリティを強化した。Next.js は事前告知付きの月次セキュリティリリース方針を発表している。

## ヘッドライン

### Release pnpm 11.10 · pnpm/pnpm
- **URL**: https://github.com/pnpm/pnpm/releases/tag/v11.10.0
- **タグ**: pnpm ReleaseNote
- **要約**: `pnpm issues`/`pnpm prefix` コマンドや `pnpm_config__auth` 環境変数、self-update での pnpm v12 サポートを追加。`pnpm up -r` が無関係なパッケージを更新する問題や `pnpm pack-app` のパス検証を修正した。

### Chrome 150 | Release notes | Chrome for Developers
- **URL**: https://developer.chrome.com/release-notes/150
- **タグ**: Chrome browser css DOM WebGPU ReleaseNote
- **要約**: CSS `AccentColor`/`AccentColorText`、polygon の角丸、`text-fit`、cross-origin/integrity/referrer-policy 用の URL modifier、DOM の `focusgroup` 属性、Promise を返すスクロールメソッド、WebGPU `setImmediateData()`、PWA のオリジン移行、実験的な `performance.getSpeculations()` を導入。

### Electron 43 | Electron
- **URL**: https://www.electronjs.org/blog/electron-43-0
- **タグ**: Electron ReleaseNote
- **要約**: Chromium 150.0.7871.46、Node.js v24.17.0、V8 15.0 へ更新。ファイルダウンロードのデフォルト先を Downloads フォルダに変更し、`WebContents.clone()`、macOS の通知削除メソッド、`globalShortcut.setSuspended()` を追加。startup snapshot と V8 バイトコードキャッシュで起動性能を改善。

### Announcing TypeScript 7.0 - TypeScript
- **URL**: https://devblogs.microsoft.com/typescript/announcing-typescript-7-0/
- **タグ**: TypeScript ReleaseNote
- **要約**: Go 製のネイティブ `tsc` コンパイラを npm でリリース。`target: es5` や `baseUrl` などの非推奨設定はエラー化。`--checkers`/`--builders`/`--singleThreaded` で並列処理を制御し、`--watch` モードを再構築、LSP のエディタサポートを改善した。Program 向け API は未公開で、Vue などのツールとの互換性は限定的。

### Better Auth is joining Vercel
- **URL**: https://better-auth.com/blog/better-auth-joins-vercel
- **タグ**: TypeScript vercel news nodejs
- **要約**: Vercel が TypeScript 製認証ライブラリ Better Auth を買収。開発は MIT ライセンスのまま継続する。Agent Auth Protocol の開発や、エージェントの ID 管理を Vercel Connect・Eve へ統合する計画がある。

### npm install-time security and GAT bypass2fa deprecation - GitHub Changelog
- **URL**: https://github.blog/changelog/2026-07-08-npm-install-time-security-and-gat-bypass2fa-deprecation/
- **タグ**: npm security ReleaseNote
- **要約**: npm v12 は `allowScripts` をデフォルト無効化し、`--allow-git`/`--allow-remote` を `none` に設定。2026年8月からは 2FA バイパス用の Granular Access Token でアカウント/パッケージ操作の 2FA をスキップできなくなり、2027年1月からは直接 publish も不可に。Trusted Publishing や Staged Publishing への移行を推奨。

### Node.js — Node.js 26.5.0 (Current)
- **URL**: https://nodejs.org/en/blog/release/v26.5.0
- **タグ**: nodejs ReleaseNote
- **要約**: `Blob.prototype.textStream()`、`--experimental-import-text`、`ReadableStreamTee`、`perf_hooks.monitorEventLoopDelay` の `samplePerIteration` オプションを追加。
