## React

### Next.js May 2026 Security Release
- **URL**: https://vercel.com/changelog/next-js-may-2026-security-release
- **要約**: DoS、ミドルウェア/プロキシバイパス、SSRF、キャッシュポイズニング、XSSを含む13件の脆弱性に対応した協調セキュリティリリース。v15.5.18またはv16.2.6へのアップデートが推奨される。React RSCパッケージ(v19.2.6)にもServer Functions DoS修正が含まれる。

### TanStack npm Supply-Chain Compromise
- **URL**: https://tanstack.com/blog/npm-supply-chain-compromise-postmortem
- **要約**: 攻撃者が5月11日に42の`@tanstack/*` npmパッケージに悪意あるバージョンを公開。Mini Shai-Huludワームが他のメンテナーに感染し、Mistral AI、OpenSearch、UiPath等のパッケージにも拡散。GitHub Actionの`pull_request_target`トリガーを悪用し、共有pnpmキャッシュに毒入りコンテンツを注入する攻撃手法。

### Hardening TanStack After the npm Compromise
- **URL**: https://tanstack.com/blog/incident-followup
- **要約**: サプライチェーンリスクを低減するためのメンテナー向けガイドライン。危険な`pull_request_target`トリガーの回避や、機密ワークフローからの共有キャッシュ除去などの対策を提示。

### npm Package Manager Security Best Practices
- **URL**: https://github.com/lirantal/npm-security-best-practices
- **要約**: npmパッケージ利用者向けの実践的なセキュリティ推奨事項を提供する標準的なリソース。各パッケージマネージャー横断での対策をカバー。

### Projecting React
- **URL**: https://tannerlinsley.com/posts/projecting-react
- **要約**: Tanner LinsleyがTanStack Start向けに最適化されたReactの「projection」を`@tanstack/redact`で実現し、大幅なサイズ削減を達成。ReactのAPIをデータベーステーブルに、実装をマテリアライズドビューに例えて解説。

### RSC Server Functions Are Not An API Boundary
- **URL**: https://longho.dev/posts/rsc-server-functions-are-not-an-api-boundary/
- **要約**: Server Functionsは安定的なAPI代替にはならない。リファクタリング時にServer Function IDが変更され、再デプロイ時にバージョンスキュー問題が発生する。クラウドプロバイダーがスキュー保護機能を提供している。

### Untangling Dialogs in React Router
- **URL**: https://programmingarehard.com/2026/05/06/react-router-dialogs.html/
- **要約**: 親の`<Outlet>`と専用ダイアログルートを使ったコード簡素化のガイダンス。データ再検証、フラッシュセッショントースト、退出アニメーションをカバー。

### Animating Container Bounds
- **URL**: https://www.userinterface.wiki/animating-container-bounds
- **要約**: Motionを使って内部コンテンツの変更に基づいて要素のwidth/heightをスムーズにアニメーションさせる手法のデモ。

### Security in React Applications
- **URL**: https://certificates.dev/blog/security-in-react-applications
- **要約**: ReactアプリケーションにおけるXSS攻撃防止、安全なトークン保存、入力バリデーション、CSRF保護、Content Security Policyについて解説。

### From React to Native Web with Nanotags: A 100 KB Savings
- **URL**: https://evilmartians.com/chronicles/from-react-to-native-web-with-nanotags-a-migration-that-saved-100kb
- **要約**: 主に静的なAstroテンプレートサイトでは、Custom Elementsハイドレーションの方がReactハイドレーションより効率的になりうることを示した事例。100KBの削減を実現。

### Exploring the HTML-in-Canvas Proposal
- **URL**: https://tympanus.net/codrops/2026/05/13/exploring-the-html-in-canvas-proposal/
- **要約**: HTML-in-Canvasが3Dワールド内の2D UI統合をどのように簡素化できるかを議論。特にReact Three Fiber開発者にとって有用。

### SVAR React Gantt
- **URL**: https://github.com/svar-widgets/react-gantt
- **要約**: インタラクティブなプロジェクトタイムラインを構築するためのオープンソースReact Ganttライブラリ。

### Waku 1.0 Beta
- **URL**: https://waku.gg/blog/waku-v1-beta
- **要約**: ミニマルなRSCフレームワークの初ベータ版。Vite 8、Rolldown、Node 26、Reactパフォーマンストラック、柔軟なルーティング、CSPをサポートし、プロダクション対応へ移行。

### Jotai 2.20
- **URL**: https://newsletter.daishikato.com/p/jotai-v2-20-0-and-the-store-building-blocks
- **要約**: 高スループットシナリオのパフォーマンス改善とストアビルディングブロックのリファクタリング。

### HTML React Parser 6.1
