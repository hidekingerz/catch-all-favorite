## 今週の注目ポイント

- **Next.js 16.3 プレビュー — Instant Navigations**: サーバー駆動ナビゲーションの体感速度を高める新機能。データ取得の挙動（Suspense によるストリーミング / `'use cache'` によるキャッシュ / ブロッキング）を明示的に選べるようになり、Playwright 用 `instant()` テストヘルパーや、per-link prefetch に代わる route shell 経由の部分プリフェッチを導入。あわせて Turbopack が dev のメモリを最大90%削減、永続キャッシュで最大5.5倍高速なリビルドを実現し、Rust 版 React Compiler も実験的に搭載。
- **Astryx — Meta のデザインシステムが OSS 化**: Meta が8年かけて構築し13000以上の社内アプリを支えるデザインシステムを OSS ベータとして公開。StyleX の atomic CSS-in-JS を基盤とし、デザイントークンで高度にテーマ可能・コンポーネントは eject 可能。StyleX 採用を検討するチームに大きな参考事例。
- **React Compiler をめぐる議論の深化**: Rspack 2.1 が SWC 経由で React Compiler をファーストクラス統合（Babel 版比で7〜13倍高速）する一方、Rolldown はバイナリサイズ増を理由に Rust 版統合を撤回。さらに「React Compiler は Retrofit（後付け改修）だ」とする論考が、バンドルサイズ増やメモ化契約の不透明化を指摘し、コンパイラよりリンターとしての利用を推奨。
- **`use()` API リファレンスの全面刷新**: React 公式ドキュメントの `use()` ページが書き直され、クライアントでの Promise キャッシュ要件などが整理された。RSC/Suspense 時代の非同期データ取得を理解する上で必読。
- **ビルド/ツールチェーンの大型更新ラッシュ**: TypeScript 7 ネイティブコンパイラへの移行で VSCode の型チェックが36秒→5秒に短縮、Node.js 26.4 が Package Maps でファイル走査を削減、Deno 2.9・Webpack 5.108・Prettier 3.9・Flow（OCaml→Rust 移植で2倍高速化）と、ネイティブ高速化の潮流が全方位で進行中。
