## 今週の注目ポイント

- **TypeScript 7.0 リリース**: Go による書き直しでプロダクション対応に。型チェックが最大16倍高速化し、デフォルトで `strict: true` になる。既存プロジェクトの移行やCIのビルド時間に大きく影響するため要チェック。
- **ES2026 が正式承認**: Ecma International が新標準を承認。`Array.fromAsync()`、`Error.isError()`、`Map.getOrInsert()` などが標準化された。
- **shadcn 4.13 で Base UI がデフォルトに**: shadcn CLI とドキュメントのデフォルトの headless ライブラリが Radix から Base UI へ移行。既存の Radix サポートは維持される。UIコンポーネント基盤に関わる変化。
- **Next.js に WebSocket 一級サポートの RFC**: App Router の Route Handlers で `NextResponse.upgrade()` を提案。Vercel が WebSocket 対応を追加し、セルフホスト構成でも動作する見込み。
- **サプライチェーンセキュリティの強化事例**: TanStack 侵害後、nuqs が npm の staged publishing（draft→validate→finalize）でリリースパイプラインを堅牢化。依存パッケージ運用の参考になる。
