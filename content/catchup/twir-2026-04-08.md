> 発行日: 2026-04-08

## 今週の注目ポイント

1. **Railway が Next.js から TanStack Start に移行** — 200以上のルートを持つ Page Router アプリを2つのPRでゼロダウンタイム移行。ビルド速度やレイアウト合成の改善が報告されており、Next.js 以外の選択肢としての TanStack Start の実用性が示された。

2. **MDN のフロントエンド刷新** — React CRA ベースの旧フロントエンドを廃止し、Lit + Custom Elements 上に独自の Server Components システムを実装。Declarative Shadow DOM によるプログレッシブエンハンスメントという興味深いアプローチ。

3. **Axios サプライチェーン攻撃の影響範囲** — 週間1億DLを超える axios がソーシャルエンジニアリング攻撃で侵害された。直接依存していなくても推移的依存やCI上のnpxで影響を受ける可能性がある。npm セキュリティベストプラクティスの見直しが推奨されている。

4. **Material UI 9.0 / Mantine 9.0 同時リリース** — 主要UIライブラリの2つがメジャーバージョンを同時にリリース。MUIはアクセシビリティ改善とsx propパフォーマンス向上、Mantineは新コンポーネント多数追加。

5. **JSIR — GoogleによるJavaScript中間表現のRFC** — Hermes バイトコードの逆コンパイルなど静的解析に使われるIR。Google内部で既に活用されており、JavaScript ツールチェーンの将来に影響を与える可能性がある。
