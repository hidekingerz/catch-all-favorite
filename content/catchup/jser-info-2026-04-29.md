## 今週の注目ポイント
Node.js 24.15.0がLTSとしてリリースされ、`require(esm)`やモジュールコンパイルキャッシュが安定版に昇格しました。
Rspack 2.0ではPure ESMへの移行やReact Server Componentsの実験的対応が追加されています。
pnpm 11もPure ESMに移行し、npm CLI依存を排除してネイティブ実装に切り替えるなど大きな変更が入りました。
Fresh 2.3ではデフォルトで非JavaScriptになり、View Transitions APIやWebSocket対応が追加されています。

## ヘッドライン

### React Aria v1.17.0
- **URL**: https://react-aria.adobe.com/releases/v1-17-0
- **タグ**: React, library, ReleaseNote
- **要約**: Tableの展開可能な行対応、Virtualizerのウィンドウスクロール対応、水平Virtualize対応。個別hookパッケージを統合して依存数削減。sub-path imports対応、codemod提供。ComboBoxのジェネリック型、NumberFieldの`commitBehavior` prop、Selectの`shouldCloseOnSelect`オプション追加。

### Node.js 24.15.0 (LTS)
- **URL**: https://nodejs.org/en/blog/release/v24.15.0
- **タグ**: nodejs, ReleaseNote
- **要約**: `require(esm)`とモジュールコンパイルキャッシュが安定版に昇格、SQLiteがリリース候補段階へ進行。`--max-heap-size`オプション、Socket `setTOS`/`getTOS`メソッド、`fs.stat`の`throwIfNoEntry`オプション追加。

### Bun v1.3.13
- **URL**: https://bun.com/blog/bun-v1.3.13
- **タグ**: Bun, ReleaseNote
- **要約**: `bun test`に`--isolate`/`--parallel`/`--shard`/`--changed`フラグ追加。`bun install`のメモリ改善、monorepo速度改善。SHA3対応、X25519鍵導出対応。WebSocketクライアントがUnixドメインソケット対応。Range対応、gzip圧縮パフォーマンス改善。

### Rspack 2.0
- **URL**: https://rspack.rs/blog/announcing-2-0
- **タグ**: rspack, ReleaseNote
- **要約**: `@rspack/core`などをPure ESMに移行、CommonJSビルド削除。ビルドパフォーマンス改善、CommonJS require分割代入のTree Shaking改善。`/*#__NO_SIDE_EFFECTS__*/`アノテーション対応、Module Federationの共有依存のExportレベルTree Shaking、`library.type`に`modern-module`追加。React Server Components実験的対応、`#/`サブパスエイリアス、swc-loader `detectSyntax`オプション追加。

### Fresh 2.3
- **URL**: https://deno.com/blog/fresh-2.3
- **タグ**: deno, vite, ReleaseNote
- **要約**: 静的ページでデフォルト非JavaScriptに変更、View Transitions APIサポート、`app.ws()`によるWebSocket対応、Vite統合改善。CSP nonce自動生成、CIDR形式IPフィルタリング、OpenTelemetry Server to Client伝播、Temporal APIサポート。複数静的ディレクトリ対応、フォーム送信ローディング表示、リバースプロキシ対応。

### pnpm 11
- **URL**: https://github.com/pnpm/pnpm/releases/tag/v11.0.0
- **タグ**: pnpm, ReleaseNote
- **要約**: Pure ESMに移行、Node.js 18-21サポート終了。`.npmrc`は認証/レジストリ限定、他の設定は`pnpm-workspace.yaml`へ移行。`minimumReleaseAge`デフォルト1日に変更、SQLiteストア使用。npm CLI依存排除して`pnpm publish`/`pnpm login`/`pnpm audit`をネイティブ実装。`pnpm ci`/`pnpm sbom`/`pnpm clean`/`pnpm pack-app`コマンド追加。
