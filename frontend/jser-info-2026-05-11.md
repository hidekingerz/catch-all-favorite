# JSer.info #771 キャッチアップ: 2026-05-11のJS

> 投稿日: 2026-05-11
> ソース: [2026-05-11のJS: Node.js 26.0.0、Rolldown 1.0、Jest v30.4.0](https://jser.info/2026/05/11/node.js-26.0.0-rolldown-1.0-jest-v30.4.0/)

## 今週の注目ポイント
Node.js 26.0.0がリリースされ、Temporal APIがデフォルト有効化された。現行LTSスケジュール下の最終メジャーリリースであり、以降は年1回リリースへ移行予定。
Rustで書かれたRollup互換バンドラーのRolldown 1.0が安定版に到達し、Vite 8でデフォルトバンドラーとして採用予定。
Jest v30.4.0ではNode v24.9+でのESMの`require()`サポートやNode v26 Temporal API対応のfake timersが追加された。
Node.js 20.xがEOLとなりセキュリティパッチの提供が終了した。

---

## ヘッドライン

### AVA v8.0.0
- **URL**: https://github.com/avajs/ava/releases/tag/v8.0.0
- **タグ**: JavaScript, testing, nodejs
- **要約**: Node.js v22.20/v24.12以降必須。内部実装をESM化し、CommonJS利用時は`require('ava')`構文の変更が必要。デフォルトテストファイル拡張子が`js/mjs`に変更、`test.skipIf()/test.runIf()`メソッド追加。

### Node.js 26.0.0
- **URL**: https://nodejs.org/en/blog/release/v26.0.0
- **タグ**: nodejs
- **要約**: Temporal APIがデフォルト有効化。V8 14.6更新により`Map/WeakMapのgetOrInsert()`、`Iteratorのconcat()`追加。`--experimental-transform-types`フラグ削除、`module.register()`のruntime-deprecate化など破壊的変更含む。

### StyleX v0.18
- **URL**: https://stylexjs.com/blog/v0.18
- **タグ**: React, css library
- **要約**: `stylex.env` APIでコンパイル時値設定可能化。`create-stylex-app` CLI追加、Chrome DevTools拡張提供。JSXで`sx={}`構文サポート、`defineVars`の自己参照変数サポート。

### Node.js 20.x EOL
- **URL**: https://nodejs.org/en/about/eol
- **タグ**: nodejs
- **要約**: Node.js 20.xがEOLとなり、セキュリティパッチを含めたアップデートが提供されなくなった。

### Chrome 148
- **URL**: https://developer.chrome.com/release-notes/148
- **タグ**: Chrome, browser
- **要約**: CSSコンテナクエリ、`@supports` `at-rule()`サポート。`text-decoration-skip-ink: all`、`video/audio`の`loading="lazy"`属性対応。Prompt API、Web Authentication Immediate UIモード追加。

### Node.js 26.1.0
- **URL**: https://nodejs.org/en/blog/release/v26.1.0
- **タグ**: nodejs
- **要約**: `node:ffi`モジュール追加で動的ライブラリ読み込みに対応。`crypto.randomUUIDv7()`、`fs.stat()`の`signal`オプション追加。HTTPの`IncomingMessage`に`req.signal`追加。

### Jest v30.4.0
- **URL**: https://github.com/jestjs/jest/releases/tag/v30.4.0
- **タグ**: jest, test
- **要約**: Node v24.9+でES Modulesに対する`require()`をサポート。Node v26 Temporal API対応fake timers、`jest.config.mts`サポート、プロジェクト単位の`verbose/silent`設定追加。React 19対応。

### Electron 42
- **URL**: https://www.electronjs.org/blog/electron-42-0
- **タグ**: Electron
- **要約**: Chromium 148/Node.js 24.15.0/V8 14.8更新。macOS通知APIが`UNNotification` APIに変更。`view.setBounds()`アニメーション対応、ネイティブ背景ぼかしサポート追加。

### axios v1.16.0
- **URL**: https://github.com/axios/axios/releases/tag/v1.16.0
- **タグ**: HTTP library, JavaScript
- **要約**: fetch adapterでの`maxBodyLength/maxContentLength`制限強制化。QUERY HTTPメソッド追加、`AxiosError.ECONNREFUSED`追加、基本認証情報デコード対応。

### Rolldown 1.0
- **URL**: https://voidzero.dev/posts/announcing-rolldown-1-0
- **タグ**: Rust, rollup, bundler, vite
- **要約**: Rustで書かれたRollup互換プラグインとの互換性を持ったJavaScript/TypeScriptバンドラー。Vite 8でデフォルトバンドラーとして採用予定。

### Chrome 149 beta
- **URL**: https://developer.chrome.com/blog/chrome-149-beta?hl=en
- **タグ**: Chrome, browser
- **要約**: CSSの`shape-outside`で`path()/shape()/rect()/xywh()`関数サポート。SVG `pathLength`属性CSS化、`Intl.Locale.prototype.variants`追加。Service Workerに`isReloadNavigation`属性追加。
